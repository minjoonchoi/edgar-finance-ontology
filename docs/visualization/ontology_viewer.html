적용 ODP 패턴

• Observation Pattern: FinancialObservation 중심 구조
• Canonicalization: 이질적 XBRL 태그 통합
• Contextualization: Period, Currency, AccountingStandard
• Property Chain: hasIndustryOfObservation (forEntity ∘ hasIndustry)
• Key Pattern: (forEntity, forMetric, hasPeriod, hasCurrency)로 유일성 보장
